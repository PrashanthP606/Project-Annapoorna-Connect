Annapoorna Connect

A full-stack web application designed to reduce food waste by connecting food donors with NGOs and people in need. This platform enables efficient food redistribution with real-time tracking and role-based access.

🚀 Features
👤 User & Admin Authentication (Role-based access)
📦 Food donation posting system
🔍 Browse and request available food
📍 Location-based coordination
📊 Dashboard for managing requests and donations
🔐 Secure backend APIs

🛠️ Tech Stack
Frontend
1. React (Vite)
2. Tailwind CSS
3. Axios

Backend
1. Node.js
2. Express.js
3. MongoDB

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/YOUR_USERNAME/Project-Annapoorna-Connect.git
cd Project-Annapoorna-Connect
2️⃣ Setup Backend
cd Backend
npm install

Create a .env file in Backend:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

Run backend:

npm start
3️⃣ Setup Frontend
cd frontend
npm install
npm run dev
